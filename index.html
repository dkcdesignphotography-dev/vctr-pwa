1. What VCTR Does

VCTR helps users decide where a prompt or task should go next. VCTR currently supports two routing modes:

  • Offline Routing: Uses keyword matching on this device/browser. No external AI call is made by VCTR.
  • Smart Routing: Uses the user's Anthropic API key to send the prompt directly to Anthropic for routing assistance.

2. Information VCTR May Store Locally

  • Your selected AI stack
  • Your routing preferences
  • Your optional Anthropic API key
  • Learned routing patterns based on your corrections
  • Basic local app settings

This information is stored on the device/browser you use to access VCTR. VCTR does not currently operate a separate user account system or central database.

3. Information That May Leave Your Device

If you use Offline Routing, VCTR routes your prompt on-device using keyword matching only.

If you use Smart Routing, the prompt you enter may be sent directly from your device to Anthropic using your Anthropic API key. Anthropic may process that prompt according to their own terms and policies.

If you open a third-party AI destination, anything you submit there is handled by that platform, not VCTR.

4. API Keys

Your Anthropic API key is your credential. Treat it like a password. VCTR stores your API key locally on your device/browser if you choose to enable Smart Routing. VCTR does not operate a server to store API keys externally. Do not save an API key on a shared, public, or untrusted device.

5. Prompts and User Content

VCTR does not claim ownership of your prompts, tasks, or ideas. VCTR does not currently store your prompts on a VCTR-operated server. Do not enter sensitive, confidential, regulated, private, financial, medical, legal, or personal information unless you are comfortable sending that information to the selected third-party platform.

6. Data We Do Not Sell

VCTR does not sell user data. VCTR does not use prompt content for advertising. VCTR does not knowingly sell, rent, or trade API keys, prompts, routing settings, or learned route data.

7. Analytics and Tracking

At this time, VCTR does not use advertising trackers or third-party ad networks. If analytics or tracking tools are added in the future, this Privacy Policy will be updated before those tools are enabled.

8. Data Controls

Users can control local VCTR data through Settings. Users may clear learned routes, clear the saved API key, reset VCTR, or use Offline Routing instead of Smart Routing. Resetting VCTR clears local settings on that device. It does not delete data already submitted to Anthropic or another third-party platform.
